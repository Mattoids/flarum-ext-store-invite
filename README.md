A [Flarum](http://flarum.org) extension. flarum store invite

## Installation

Install with composer:

```sh
composer require mattoid/flarum-ext-store-invite:"*"
```

## Updating

```sh
composer update mattoid/flarum-ext-store-invite:"*"
php flarum migrate
php flarum cache:clear
```
